# 使用ES6模块
需要创建`package.json`文件
```
npm init -y
```
添加
```json
"type":"module"
```

## 例如
```json
{
  "name": "moudle",
  "version": "1.0.0",
  "description": "",
  "main": "main.js",
  "type":"module",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": [],
  "author": "",
  "license": "ISC"
}
```
